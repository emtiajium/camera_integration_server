## Prerequisite
Please make sure you have working setup of these tools before trying it. 
 
 1. Node with npm
 2. Bower
 
## Installation
In order to use this boilerplate, please follow these steps

 1. clone project
 2. `bower install`
 3. `npm install`
 
 
To run the app do  
```
npm start 
```

Now visit the following url in your browser,
```
http://localhost:9000
```
